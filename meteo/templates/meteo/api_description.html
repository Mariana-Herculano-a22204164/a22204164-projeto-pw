API Meteo

Esta API fornece informações meteorológicas para várias cidades.

Endpoints

  • Lista de Cidades

    URL: /meteo/api/cities/

    Método: GET

    Descrição: Retorna a lista de cidades para as quais existem previsões meteorológicas.

  • Previsão Meteorológica de Hoje para uma Cidade

    URL: /meteo/api/weather/today/<city_id>/

    Método: GET

    Descrição: Retorna a previsão meteorológica de hoje para a cidade especificada.

    Parâmetros:

    • city_id: O ID da cidade.

  • Previsão Meteorológica para os Próximos 5 Dias para uma Cidade

    URL: /meteo/api/weather/next-5-days/<city_id>/

    Método: GET

    Descrição: Retorna a previsão meteorológica para os próximos 5 dias para a cidade especificada.

    Parâmetros:

    • city_id: O ID da cidade.
